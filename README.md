# Rust Notepad

Windows-first plain text editor in Rust using native Windows UI components.

## Current Status

This repository currently contains the environment and project setup only.

## Planned Stack

- Rust
- Native Windows GUI
- Standard library file I/O

## Local Development

1. Install Rust with `rustup`.
2. Open this folder in VS Code.
3. Install the recommended workspace extensions.
4. Run `cargo check` once Rust is available on `PATH`.
